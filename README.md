## рҹ“Ң Overview

FuturePath AI is a Machine Learning-based Career Prediction System designed to guide students toward the most suitable career path. The system analyzes multiple parameters such as CGPA, Coding Skills, Communication Skills, Aptitude Level, and Interest Area to generate intelligent career recommendations.

This project demonstrates a complete Machine Learning workflow, including dataset handling, preprocessing, model training, evaluation, and prediction through a modern web-based interface.

## рҹҺҜ Project Objective

Many students struggle to identify the career path that best matches their abilities and interests.
